# Contributing

Thanks for your interest in improving Jenkins Client.

## Development Setup

```bash
npm install
npm run compile
npm run lint
```

Open this folder in VS Code and use the **Launch Extension** debug configuration to run the extension in an Extension Development Host.

## Pull Requests

- Keep changes focused and easy to review.
- Run `npm run compile` and `npm run lint` before opening a pull request.
- Include details about your Jenkins job type when changing Jenkins API behavior.
- Update the README when adding or changing user-facing settings or commands.
